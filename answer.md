# 外骨骼与可穿戴机器人技术的进展与应用

## 研究背景
外骨骼与可穿戴机器人技术在康复和辅助行走领域展现出重要的应用潜力。这些系统通过提供机械助力、调节运动轨迹以及优化能量效率等方式，改善了因神经系统损伤或肌肉功能障碍导致的行走障碍。近年来，软体外骨骼（soft exosuits）因其轻便性、灵活性和对生物关节的兼容性，逐渐成为研究热点。文献中报道了多种软体外骨骼的设计与控制策略，旨在提升行走过程中的力矩输出、步态对称性和能量消耗效率。与此同时，针对不同人群的个性化需求，研究者也在探索自适应控制方法与实时反馈机制。综上所述，外骨骼与可穿戴机器人技术的发展不仅为康复医学提供了新的解决方案，也为日常行走辅助设备的设计与优化带来了新的思路。

- 为改善因脑瘫导致的屈膝步态，研究提出了基于双膝外骨骼的控制方法，通过提供膝关节伸展力矩来减少异常膝关节屈曲，并采用轨迹控制以增强膝关节屈伸运动【2】。
- 针对中风后遗症患者，文献中介绍了采用软体外骨骼进行临床步态训练的方法，该系统通过机械助力提升患者踝关节的跖屈与背屈功能，从而改善行走过程中的推进对称性与足清空度【4】。
- 在自主多关节软体外骨骼的研究中，文献提出了基于行走周期的力矩控制策略，并通过传感器系统实时采集数据以调整控制参数，从而提高行走过程中力的稳定性【1】。
- 为适应不同地形条件，研究开发了具有自适应控制能力的软体外骨骼系统，能够在不平整地面上保持力的输出一致性，减少峰值力偏差并降低均方根误差【1】。
- 为了实现自然步态的引导与协助，文献提出了基于虚拟流场的控制方法，通过影响腿部运动轨迹来改善中风患者的行走平衡与步态协调性【5】。

外骨骼与可 wearable 机器人技术的发展正在逐步满足个性化康复与辅助行走的需求，其在提升行走效率、改善步态对称性及增强用户适应性方面展现出广阔的应用前景。

## 系统设计与关键技术
外骨骼/可穿戴机器人系统设计与关键技术是实现高效、安全和舒适的人机交互的核心。系统设计需综合考虑动力传输、控制策略、用户输入响应以及能源管理等多个方面，以满足不同应用场景下的性能需求。在控制方面，研究者提出了多种控制方法，如开环控制与闭环控制的结合、基于回归模型的控制策略等，旨在提升系统的实时性与适应性。以下从几个关键方面进行分析：

1. **开环与闭环控制策略的对比**：开环控制因其结构简单、成本低廉而被广泛应用于消费级外骨骼设备中，但其对用户输入的响应能力有限。研究表明，基于广义线性回归的开环控制框架在踝关节助力方面表现优异，其根均方扭矩跟踪误差与闭环扭矩反馈控制相当，同时具有更低的能耗和更少的运行噪声【1】。相比之下，闭环控制通过实时反馈实现更精确的力矩调节，但对传感系统的要求更高，且存在潜在的延迟问题【1】。

2. **基于统计建模的控制方法**：为了提高控制系统的适应性，研究者提出了基于统计建模的控制方法，例如利用踝关节角速度和目标力矩作为输入变量的“复杂”模型，或仅使用目标力矩的“简单”模型。这种基于回归的控制策略不仅提升了控制精度，还在降低能耗方面表现出显著优势【1】。此外，通过结合力矩误差、平均与峰值力矩输出等指标，可以进一步优化控制模型的性能【1】。

3. **软体外骨骼的创新设计**：与传统刚性外骨骼相比，软体外骨骼采用柔性材料与轻量化结构，提高了穿戴舒适度与运动自由度。例如，一种基于麦金布恩气动执行器的软体下肢外骨骼，通过虚拟锚点技术实现力的高效传递，其总质量仅为3.5 kg，并在被动模式下对关节运动学影响较小【4】。此外，软体外骨骼还通过实时监测关节运动学与代谢功率，实现对用户活动的自适应控制，从而降低代谢成本【4】。

4. **传感与数据采集系统的集成**：为了实现精准控制，外骨骼系统依赖于多种传感器，包括IMU（惯性测量单元）、力传感器（如力传感器）以及肌电信号（EMG）等。这些传感器采集的实时数据不仅用于力矩控制，还支持相位识别与运动状态监测，以确保系统能够根据用户的动作动态调整输出【1,4】。例如，在踝关节助力系统中，踝关节速度与力的实时测量对于优化控制策略至关重要【1】。

5. **个体自适应与能量效率**：外骨骼系统的性能不仅依赖于硬件设计，还需结合个体差异进行自适应优化。例如，通过调整控制参数，可以提高系统在不平整地形上的力矩一致性，从而减少力矩误差并提升行走稳定性【4】。此外，研究还表明，合理的控制策略能够显著降低能耗，提高行走效率，从而改善用户的代谢功率与整体运动表现【4】。

综上所述，外骨骼/可穿戴机器人系统的系统设计与关键技术涵盖控制策略、传感集成、个体自适应以及能量管理等多个方面，这些技术的综合应用将推动外骨骼在医疗康复、消费电子以及工业应用等领域的进一步发展。

## 性能评估与测试方法
外骨骼和可穿戴机器人系统的性能评估通常涉及对运动学、动力学以及控制策略的测试，以确保其在实际应用中的有效性与安全性。这些测试方法包括但不限于基于惯性传感器的数据采集、力矩控制的验证以及运动状态的实时监测。为了准确评估系统性能，需要结合多种技术手段进行综合分析。

1. **惯性传感器与运动捕捉系统的对比**：惯性运动传感技术通过MEMS（微机电系统）传感器采集人体运动数据，具有便携性和无需视线遮挡的优势，适用于动态环境下的实时监测。然而，其在运动学参数估计的准确性方面仍需与光学运动捕捉系统进行对比验证。例如，在对深蹲动作的评估中，惯性传感器测量得到的正常地面反作用力（GRF）与力板测量（地面真实值）的相关性达到0.93，RMSE小于0.02倍体重，表明其具备足够的精度用于逆动力学分析【1】。

2. **控制策略与力矩输出的评估**：在设计可穿戴机器人时，控制策略的性能通常通过力矩输出的稳定性与响应速度来衡量。例如，多盘制动器通过压缩摩擦盘产生阻尼力矩，其动态范围、带宽和电气功耗等指标被用于评估其性能【2】。此外，比例制动器能够在不同模式下切换，以适应不同的应用需求，如在电力故障时切换至锁定模式，提高系统的安全性【2】。

3. **个体自适应与临床性能指标**：在临床应用中，外骨骼与可穿戴机器人需要根据患者个体特征进行自适应调整，以优化其功能表现。例如，可变刚度假肢踝关节的首选刚度与患者的步态速度和代谢成本密切相关，且在接近首选刚度范围内调整时，不会显著影响代谢成本【3】。这表明，临床性能指标（如步态速度、代谢成本）可以作为评估个体自适应性能的重要依据。

4. **实时性与系统延迟的分析**：在实时控制中，系统延迟对运动同步性和安全性具有重要影响。例如，在使用软外骨骼进行踝关节辅助时，需要对踝速和力的实时测量进行分析，以确保控制策略的响应速度满足人体运动需求【4】。此外，系统延迟还可能影响力矩控制的精度，因此需要通过实验数据进行量化评估。

综上所述，外骨骼与可穿戴机器人系统的性能评估需要结合多种测试方法，包括惯性传感器与光学运动捕捉系统的对比、控制策略的力矩输出验证、个体自适应能力的临床评估以及系统实时性与延迟的分析，以确保其在实际应用中的有效性与安全性。

## 总结与展望
外骨骼与可穿戴机器人技术作为人机交互的重要领域，正在不断拓展其在医疗康复、辅助行走及智能穿戴等场景中的应用。当前研究已涵盖系统设计、控制策略、性能评估等多个方面，形成了较为完整的理论与实践体系。未来，随着材料科学、人工智能与生物力学的进一步融合，外骨骼技术有望实现更高水平的个性化适配与智能化控制。同时，如何在保证功能性能的同时提升系统的舒适性与安全性，将是推动该技术走向更广泛应用的关键方向。
